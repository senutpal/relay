# frontend

react 19 spectator interface for live sports commentary.

## tech stack

- react 19 with typescript
- vite
- tailwind css v4
- tanstack query
- websocket client

## setup

```bash
npm install
npm run dev
```

runs on port 5173 with proxy to backend on 3001.

## features

- match list view
- live commentary feed
- real-time websocket updates
